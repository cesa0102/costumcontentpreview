============
Installation
============

Dependence
==========

Make sure, that css_styled_content is installed.

Step by step
============

Install the extension using the extension manager
Switch to the Template module and select your root page
Click on Edit the whole template record
Switch to the tab Includes
Click on TypoScript of EXT:contentmorelink (contentmorelink) inside the Available Items box
